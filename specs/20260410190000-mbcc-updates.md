## 4. "Register Now" Button — Pulsing Glow Ring Effect

### Decision
**Gold + white alternating pulsing rings.** Two concentric rings animate outward from the button border at different speeds: an inner gold ring (matching saffron brand colour) and an outer white ring. The button itself does not move.

### Locations to update
Both button instances on `src/pages/events/[slug].astro`:
1. **Hero header button** (lines ~116–128) — smaller button in the event details header
2. **Sidebar button** (lines ~280–292) — full-width sticky sidebar button

### Implementation
Define a CSS keyframe animation (e.g. `@keyframes pulse-ring`) that scales a pseudo-element from 100% to 130–150% opacity 0. Add two pseudo-elements (or two box-shadow layers at different phases) for the dual-ring effect.

**Suggested approach using `box-shadow` animation:**
```css
@keyframes pulse-ring {
  0%   { box-shadow: 0 0 0 0 rgba(245,158,11,0.6), 0 0 0 0 rgba(255,255,255,0.3); }
  70%  { box-shadow: 0 0 0 10px rgba(245,158,11,0), 0 0 0 18px rgba(255,255,255,0); }
  100% { box-shadow: 0 0 0 0 rgba(245,158,11,0), 0 0 0 0 rgba(255,255,255,0); }
}
```
- Inner ring: saffron/gold `rgba(245,158,11,...)` — aligns with `saffron-500` brand colour
- Outer ring: white `rgba(255,255,255,...)`
- Animation duration: ~2s, `ease-out`, `infinite`
- Respect `prefers-reduced-motion`: wrap in `@media (prefers-reduced-motion: no-preference)` so the animation only runs when the user has not opted out of motion.

### Sizing note
Keep the existing button sizes — the pulsing ring effect provides enough visual attention without enlarging the button.

## 5. Registered Organisations Page

### Overview
A new searchable/filterable page listing all organisations that have registered with MBCC through the Wesak grassroots registration process. This is distinct from the 6 national-level Member Organisations.

### Data Source & Import
- **Source file:** `src/assets/files/wesak-grassroot-registration.xls`
- **Strategy:** One-time import → converted to a static JSON file (e.g. `src/data/registered-organisations.json`)
- **Fields to extract from XLS:**
  - Organisation name — English (primary display)
  - Organisation name — Malay / Bahasa Malaysia (fallback if English absent)
  - Organisation name — Chinese / 中文 (always shown as subtitle)
  - State / location (Malaysian state)
  - Registration number

### Data Format (JSON)
```json
[
  {
    "nameEn": "Lotus Buddhist Society",
    "nameMs": "Persatuan Buddhis Lotus",
    "nameZh": "莲花佛学会",
    "state": "Selangor",
    "registrationNo": "MBCC-2026-001"
  },
  ...
]
```

**Name display logic:**
- Primary: `nameEn` if non-empty, else `nameMs`
- Subtitle: `nameZh` (always shown in smaller text below, if present)

### Page Details

**URL:** `/registered-organisations`  
**Page title:** "Registered Organisations — MBCC"

**Layout:** Compact table (not card grid)
```
┌──────────────────────────────────────────────────────────┐
│  [ 🔍 Search organisations...         ] [ State ▼ ]      │
├──────────────────────────────────────────────────────────┤
│  Organisation Name          │  State      │  Reg. No.    │
├──────────────────────────────────────────────────────────┤
│  Lotus Society              │  Selangor   │  MBCC-001    │
│  莲花佛学会                   │             │              │
├──────────────────────────────────────────────────────────┤
│  Dharma Centre              │  Penang     │  MBCC-002    │
│  法轮中心                    │             │              │
└──────────────────────────────────────────────────────────┘
```

**Search:** Text input that filters rows client-side (JS). Matches against English name, Malay name, Chinese name, state, and registration number. Instant, no page reload.

**Location filter:** Dropdown listing unique states derived from the data. Default: "All States". Selecting a state further filters the displayed rows (combined with any active search text).

**Pagination:** None — render all rows. (Acceptable for a static HTML page; revisit if list exceeds ~2000 entries.)

**No detail page** per organisation — the listing is the final destination.

**No count display.**
